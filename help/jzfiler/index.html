JzFiler Help

始めに

JzFilerとはJava製のテキストエディタです。
VzEditorライクなテキストエディタを目指して作成しています。
実行にはJavaSE7が必要です。

コマンドライン

F12でコマンドラインのタブが開きます。
ESCでファイルオープン
Shift+ESCでバックログモードの切り替え
バックログモードでファイル名(行番号)という書式の行でタグジャンプ

ファイラー

ファイラーは通常モードとVzライクモードがあります。
通常モードは普通のOS付属のファイラーです。

VzライクファイラーはVzのような2画面のファイルリストを表示するファイラーモードです。


TABでウィンドウの移動
Dでドライブ変更
ESCでファイラー終了
